# GLE Components
A basic React component library.

## Documentation
[Storybook](https://guyettinger.github.io/gle-components/)

## Installation
```shell
npm install gle-components@latest
```

## Development
Install
```
npm install
```
Build Library
```
npm run build
```
Run Tests
```
npm run test
```
Run Storybook
```
npm run storybook
```
